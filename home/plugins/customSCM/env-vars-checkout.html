The following variables are available to shell scripts during the checkout phase:
JOB_NAME
The name of the job
JOB_DIR
The root directory of the job
WORKSPACE
The workspace directory of the build
BUILD_NUMBER
The build number
BUILD_DIR
The root directory of the build
REVISION_STATE_FILE
Full path where the revision state file can be written (includes file name). This is optional during the checkout phase. If you write this file during checkout phase, please don't write it again during calculate revision phase. This file is not meant to be read. It will not exist at this point in time before you open it for writing. This information will persist for the compare revision phase.
OLD_REVISION_STATE_FILE
Full path of the revision state file present during the polling phase (includes file name). This file is not meant to be written to. Changes don't persist.
CHANGELOG
Full path where the file "changelog.xml" can be written (includes file name)
HUDSON_URL or JENKINS_URL
Hudson/Jenkins base URL (empty if not set by user, please set one in Email Notification configuration)
HUDSON_HOME or JENKINS_HOME
Path of Hudson/Jenkins home directory
BUILD_URL
The URL of this build
BUILD_URL_LAST
The URL of the last build (empty if none)
BUILD_URL_LAST_SUCCESS
The URL of the last successful build (empty if none)
BUILD_URL_LAST_STABLE
The URL of the last stable build (empty if none)
UPSTREAM_JOB_URLS
All upstream job URLs in order (empty if none)
